# Manual de Usuario: Guía de Operación HMO Auditor

Bienvenido al sistema **HMO Auditor**. Este manual le guiará a través del proceso de certificación utilizando inteligencia artificial local para automatizar su auditoría interna.

## 1. Inicio: Selección del Entorno
Al abrir el aplicativo, se le presentarán dos opciones:
- **Modo Simulación**: Utilice este modo para explorar el sistema con la empresa ficticia `Innovatech Solutions`. Todos los documentos ya están cargados.
- **Modo Certificación (Nueva Empresa)**: Utilice este modo para iniciar un proceso real. Deberá completar el **Checklist de Ingesta** para cargar la identidad de su organización.

## 2. Paso 1: Ingesta Guiada y Cartas de Navegación
Para garantizar que la base de conocimientos sea 100% exacta, el sistema solicita los documentos de forma secuencial:
1. **Solicitud de Documento**: El sistema le pedirá un archivo específico (ej: Misión y Visión).
2. **Carta de Navegación**: Antes de cargar, el sistema despliega una **Carta de Navegación** que explica:
    - La **Referencia ISO** que justifica el documento.
    - La **Justificación Técnica** de por qué es vital para la auditoría.
    - Los **Requisitos de Validez** que debe cumplir el archivo.
3. **Carga y Validación (OCR)**: Arrastre el archivo. En la pantalla de **"Doble Vista"**, verifique el texto extraído.
4. **Avance en Tiempo Real**: Al aprobar cada documento, el **Mapa de Nodos** se actualiza y el sistema libera la siguiente solicitud.

## 3. Paso 2: Seguimiento por Nodos
- Observe el **Mapa de Nodos** en el Dashboard. 
- Los nodos **Azules** le indicarán qué acción debe realizar a continuación (ej: Crear el Programa de Auditoría).
- Haga clic en un nodo para ver qué requisitos de la norma ISO está cubriendo.

## 4. Paso 3: Generación de Formatos Protegidos
1. Una vez que el sistema tenga suficiente información, podrá generar documentos.
2. Revise el borrador sugerido por la IA.
3. Modifique los campos necesarios (el sistema resaltará en azul lo que él cree que es correcto).
4. Presione **"Emitir Documento Oficial"**. El sistema le entregará archivos Word, Excel o PDF que **no pueden ser alterados estructuralmente**, garantizando su validez ante auditores externos.

## 5. Paso 4: Cierre y Reporte de Integridad
- Al finalizar el ciclo, descargue su **Reporte de Validación de Integridad**.
- Este reporte contiene los códigos (Hashes) que demuestran que sus documentos son veraces y no han sido manipulados.
